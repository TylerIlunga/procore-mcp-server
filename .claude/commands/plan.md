Create an execution plan for complex multi-PR work.

Follow the workflow in `.claude/skills/plan.md`:
1. Assess the full scope of what needs to change
2. Check `docs/exec-plans/active/` for related work in progress
3. Copy `docs/exec-plans/TEMPLATE.md` to `docs/exec-plans/active/YYYY-MM-DD-descriptive-name.md`
4. Define clear, testable acceptance criteria
5. Break into phases — each phase produces a mergeable PR
6. Identify risks and dependencies
7. Add an entry to `docs/PLANS.md`

Ask for the goal or feature description before starting.
