## Template

Create new specs as: `docs/product-specs/feature-name.md`

```markdown
# Feature: [Name]

**Status:** Draft | Approved | In Progress | Shipped
**Priority:** P0 | P1 | P2 | P3
**Owner:** [name]

## User Story
As a [persona], I want to [action] so that [benefit].

## Acceptance Criteria
- [ ] [Criterion]

## Design
[Link to design doc or mockups if applicable]

## Edge Cases
- [Case 1]

## Out of Scope
- [What this feature explicitly does NOT include]
```
