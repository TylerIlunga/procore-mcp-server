## How to Create a Plan

For work spanning multiple PRs or requiring coordination:

1. Create a new file in `docs/exec-plans/active/YYYY-MM-DD-plan-name.md`
2. Use the [plan template](./exec-plans/TEMPLATE.md)
3. Add an entry to this index
4. Update progress as work proceeds
5. Move to `completed/` when done
